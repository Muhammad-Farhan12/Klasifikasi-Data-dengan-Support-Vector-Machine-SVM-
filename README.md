# Tugas Praktik: Klasifikasi Data dengan Support Vector Machine (SVM) 

Repositori ini dibuat untuk memenuhi tugas mata kuliah Fundamen Sains Data. Di dalamnya terdapat implementasi *Machine Learning* menggunakan Python di Google Colab untuk mengklasifikasikan data.

##  Langkah-Langkah yang Dilakukan:
1. **Load Dataset:** Memuat dataset yang digunakan untuk klasifikasi.
2. **Preprocessing Data:** Membersihkan data dan mengubah format data agar siap dibaca oleh model (termasuk ekstraksi fitur / *scaling*).
3. **Split Data:** Membagi data menjadi *Training Set* dan *Testing Set* (80:20).
4. **Modeling SVM:** Membangun model prediksi menggunakan algoritma Support Vector Machine dari library `scikit-learn`.
5. **Evaluasi Model:** Menghitung performa model menggunakan metrik *Accuracy, Precision, Recall*, dan *F1-Score*.
6. **Visualisasi:** Menampilkan grafik distribusi kelas dan *Heatmap Confusion Matrix*.

##  Tools yang Digunakan:
* **Environment:** Google Colab
* **Bahasa:** Python
* **Library Utama:** Pandas, Scikit-Learn, Matplotlib, Seaborn
